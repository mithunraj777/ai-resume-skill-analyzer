# AI-Based Resume Skill Analyzer

An AI-powered resume analyzer that compares real resumes (PDF/DOCX) with job roles and performs explainable skill-gap analysis similar to an ATS system.

The application extracts technical skills from resumes, compares them with role-based skill requirements, and generates a match score along with missing skills.

---

## Features

- Upload real resumes (PDF / DOCX)
- Automatic skill extraction from resumes
- Role-based skill matching
- Match score calculation
- Missing skill identification
- Skill normalization (Node.js, C++, CI/CD, etc.)
- Interactive Web UI
- Progress bar visualization
- File upload preview with confirmation

## How It Works

1. Upload your resume (PDF/DOCX)
2. Select a job role
3. The system extracts skills from the resume
4. Extracted skills are compared with role-based skill requirements
5. A match score is calculated
6. The system displays:

- Match percentage
- Matched skills
- Missing skills
